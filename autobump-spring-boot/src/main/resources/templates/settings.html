Welcome to AutoBump!

Autobump is a bot that automatically creates pull-requests to keep dependencies up to date. It scans the pom.xml files, and automatically detects out-of-date dependencies using Maven central. It then creates a pull-request for every outdated dependency.


You can use this website to configure the settings of Autobump for your workspace. Select the repositories for Autobump, configure a cron job, assign reviewers and select the dependencies you wish not to be bumped.

Please select the repositories you want automatic bumps for, and configure the settings for each repository
