Welcome to AutoBump!

Autobump is a bot that automatically creates pull-requests to keep dependencies up to date. It scans the pom.xml files, and automatically detects out-of-date dependencies using Maven central. It then creates a pull-request for every outdated dependency

You can use this website to configure the settings of Autobump for your workspace:

  • Select the repositories for which Autobump must be configured
  • Select Autobump as a cron job
  • Assign a reviewer
  • Select the branches that need to be kept up to date
  • Select whether any dependencies need to be ignored

In the follow-up screen, you may define which dependencies Autobump should ignore for each repository
